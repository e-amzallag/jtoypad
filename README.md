# JToyPad
JToyPad is a Java API for the LEGO Dimensions toypad.

See [dajlab.org/jtoypad.html](https://www.dajlab.org/jtoypad.html) for more information.

## Current limitations
- Does not support writting tags.

Feel free to send pull requests.
